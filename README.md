# events

## Start the mock server

 ```bash
 npm install json-server -g
 ```
 
 Change directory into the mock-json folder

 ```bash
 json-server --watch events.json
 ```
 
 ## Start the UI
 
 Change directory into the training-events folder

 ```bash
 npm install
 
 npm start
```

The project will run on

http://localhost:4200/
